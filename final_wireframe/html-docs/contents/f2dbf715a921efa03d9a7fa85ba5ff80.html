WireframeBarricade

UMLClass
Wireframe :: Model :: WireframeBarricade

Description

Before an element is altered, the corresponding movement method is called in this class. For setting width/height/location, it is checked in this barricade class. This class calls upon the elements in the Canvas to determine overlapping elements (bad for things like text boxes).

When moving an element, it is either moved based on this information, or it will stay in place if the move is invalid. If the element can't be added to the canvas, it simply does not get added.

invalid dimensions/movements/sizes will be logged. probably though a new GUI window.

All placement will additionally call upon the canvas dimensions to make sure they fit on the screen.
