setHeight

UMLOperation
Wireframe :: Model :: WireframeBarricade :: setHeight

Description

Iterate through all elements on the canvas to see if this new change overlaps anything deemed not overlappable (text, for example). Checks if the element is locked before doing the above.

If the new change is valid, call a helper method to execute the change. This will loop through the element's annotations and set their position to this new change (not sure about changing width, for example, but definitely for changing position).
