Element

UMLClass
Wireframe :: Model :: Element

Description

Element has all the LinearProperty methods to go along with the Law of Demeter. These methods just call the getters/setters on LinearProperty.

Updating the location will call its corresponding method in the barricade, which checks if any overlapping elements are present (by iterating through each element in "THIS" element's group).

If a movement is valid, the annotation(s) also move with the element to its desired position.
